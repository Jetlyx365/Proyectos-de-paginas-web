Hardware de redes

A partir de las computadoras se necesitan axesorios, para armar las redes informaticas:

1) Placas de red: Estas placas de red permiten conectar el envio y resepción de datos transmitidos desde y hacia el equipo. En las PC moviles, viene integrada la placa madre. Se agrega las de escritorio.

2) Modem (Modulador - demodulador): Este dispositivo es necesario cuando se quiere conectar una computadoracon otra distante, utilizando algun medio de comunicación que no sea digital, como telefono o cable. Al salir los datos de la PC se deven transformar en señal analogica para que se transmita por la linea telefonica o cable, a esto se llama modulación.

El proceso inverso, es decir cuando llegan los datos coom señal analogica y deve ser transformada en digital para que la computadora lo procese; se llama demodulación.

3) Hubs (Consentradores): Este dispositivo tiene varios puertos, todos de entrada y/o salida. Estos puertos permiten conectar distintas computadoras a una red LAN. Este dispositivo esta en desuso, fue sustituido por las amplias prestaciónes de los SWITCH.

4) Switch (Computadores): Este hace la misma función que el consentrador. Se diferencian en que posee memoria. La ubicación de la información se almacena en esta memoria. Si se recibe un paquete nuevo y coinside la ubicación con una existente, se dirige directamente al equipo destino. De esta forma, se evita que halla trafico de datos innecesario.

5) Routher (Enrutador) Es más complejo que el consentrador y el conmutador. Se utiliza para conectar 2 redes que se encuentran graficamente muy distantes. Posee memoria y un Software que le permite elegir a destino.
